Tile Based Collision Detection

Overview

Tile based collision has been used since the early days of game design. \nProviding a low cost method to perform collision detections with the world layout. \nThis example aims to replicate that functionality and efficency.

Controls

Important: To start/play, press the grey box to the right. The Example when pause when section or window is pressed.

Movment

Leftward - Press A or Left Arrow Key

Rightward - Press D or Right Arrow Key

Upward - Press U or Up Arrow Key

Downward - Press S or Down Arrow Key. The player will float if in mid-air
